## Features

- Create or Delete Account
- Login (JWT Authenticated)
- Add or Delete Post
- See Your Posts
- View Latests Post from the People you follow
- Update and Edit Profile
- Search for other Users
- Follow other Users
- See Following/ Follower List/Count
